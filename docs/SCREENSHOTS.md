# Screenshots Guide

## Required Screenshots

To complete the documentation, please add the following screenshots to this directory:

### 1. Dashboard View (`dashboard.png`)
- Screenshot of the main dashboard
- Should show:
  - Real-time statistics
  - Transaction list
  - Live updates

### 2. Transaction Analysis (`analysis.png`)
- Screenshot of the analysis interface
- Should show:
  - Transaction input form
  - Risk assessment results
  - SHAP explanations (if visible)

### 3. Audit Log (`audit.png`)
- Screenshot of the audit log page
- Should show:
  - Flagged transactions
  - Transaction details
  - Review interface

### 4. Fraud Time Machine (`time-machine.png`)
- Screenshot of the temporal heatmap
- Should show:
  - Time-based fraud patterns
  - Heatmap visualization

### 5. Model Evaluation (`evaluation.png`)
- Screenshot of evaluation results
- Can be terminal output or web interface
- Should show:
  - Precision/Recall metrics
  - Confusion matrices
  - Threshold comparisons

## How to Take Screenshots

1. Run the application: `python app.py`
2. Navigate to each page
3. Take screenshots using:
   - Windows: `Win + Shift + S`
   - macOS: `Cmd + Shift + 4`
   - Linux: Use screenshot tool
4. Save images as PNG format
5. Place in `docs/screenshots/` directory
6. Update README.md paths if filenames differ
